# Student Sleep Disorder Risk Prediction

**ACM 476 — Data Mining Term Project**
Yeditepe University

## About

This project predicts a person's **sleep disorder risk** (Healthy / Mild / Moderate / Severe) based on lifestyle and health data such as sleep duration, stress score, caffeine intake, screen time, and exercise.

Dataset: 100,000 records, 32 features (`Students.xlsx`).

## Technologies Used

- Python (pandas, NumPy)
- Matplotlib, Seaborn — visualization
- scikit-learn — Decision Tree, Random Forest, model evaluation

## Project Structure

EDA/                 # Exploratory data analysis & visualizations
Decision-Tree/        # Decision Tree model
Random-Forest/         # Random Forest model
Model-Comparison/      # Accuracy / Precision / Recall / F1 comparison
